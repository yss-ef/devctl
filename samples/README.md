# Samples

This directory contains examples of projects generated using `devctl`. These
samples demonstrate the standardized architecture, configuration, and security
boilerplate implemented by the CLI.

## Contents

- `sample-api`: A Spring Boot application featuring JWT-based security and a
  Docker Compose setup for PostgreSQL or MySQL.
- `sample-front`: An Angular application featuring a pre-configured
  development proxy and environment-specific configurations.

## Usage

These folders are provided for demonstration purposes. To generate a new
project, use the `devctl init` command as described in the main documentation.
