# Samples

This directory contains examples of projects generated using `devctl`. These samples demonstrate the standardized architecture, configuration, and security boilerplate injected by the CLI.

## Contents

- **sample-api**: A Spring Boot application featuring:
    - JWT-based Security configuration.
    - PostgreSQL/MySQL ready Docker Compose setup.
    - Custom JPA properties.
- **sample-front**: An Angular application featuring:
    - Pre-configured development proxy for backend integration.
    - Standardized directory structure.
    - Environment-specific configurations.

## How to use
These folders are for demonstration purposes. To generate your own projects, use the `devctl init` command as described in the main [README](../README.md).
