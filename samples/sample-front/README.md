# Sample front

This project was generated using the Angular CLI. It serves as a demonstration
of the frontend structure created by `devctl`.

## Development server

To start a local development server, run:

```bash
ng serve
```

Navigate to `http://localhost:4200/` to view the application. The page
reloads automatically when you modify source files.

## Code scaffolding

You can use the Angular CLI to generate new components and other resources:

```bash
ng generate component component-name
```

## Building

To build the project, run:

```bash
ng build
```

Build artifacts are stored in the `dist/` directory. Production builds are
optimized for performance.

## Running unit tests

To execute unit tests with the Vitest test runner, use the following command:

```bash
ng test
```
